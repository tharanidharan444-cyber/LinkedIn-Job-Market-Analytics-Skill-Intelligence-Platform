# LinkedIn-Job-Market-Analytics-Skill-Intelligence-Platform
Companies continuously post job openings with varying skill requirements. Students often struggle to identify the most in-demand skills and career trends. This project analyzes job market data and provides actionable insights through dashboards and a skill recommendation engine.

Analyze job market trends
Identify top hiring companies
Discover most demanded skills
Compare salary distributions
Recommend missing skills to users
